# talk-pyhep-dev-2023
Introductory talk at PyHEP.dev 2023

Viewable online [here](https://matthewfeickert-talks.github.io/talk-pyhep-dev-2023/)

## Use nodeenv

```
nodeenv .env
. .env/bin/activate
npm install  # Install from package-lock.json
```
